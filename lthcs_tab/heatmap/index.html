About the Sector Heatmap

Each cell is one ticker in the LTHCS universe. Cells are colored by score band (Elite, High, Constructive, Monitor, Weakening, Review) and grouped by GICS sector. The sector header shows the name count and average score.

This is not a Finviz market-cap treemap. The LTHCS pipeline does not currently carry market-cap data per ticker, so every cell is the same size. Sectors are ordered by ticker count (largest first, ties broken alphabetically), counting only tickers visible under the current filters.

Click any cell to open the card view in a new tab with that ticker pinned in the URL fragment (e.g. ../#ticker=AAPL). The card view doesn't yet auto-open the detail modal from that fragment — that's a planned follow-up.

Data sources: ../../data/lthcs/snapshots/<latest>.json for scores, ../../data/lthcs/universe.json for sector + index membership.
