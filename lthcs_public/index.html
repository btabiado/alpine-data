What this is. A discoverable index of the JSON files this site already publishes under /data/lthcs/. External tools, notebooks, and downstream integrations can read these endpoints directly — no API key, no rate limit, no auth. The site is a static GitHub Pages mirror; every link below resolves to a plain JSON file on disk.

What to start with. Most consumers want latest_snapshot.json (stable filename, always today’s full universe scores) or manifest.json (this index, machine-readable).
